Input variables on the Mantel-Haenszel dialog box

α
The Type I error probability for a two sided test of the null hypothesis that the true within-stratum odds ratio equals 1. This is the probability that we will falsely reject the null hypothesis.

power
The probability of correctly rejecting the null hypothesis that the within-stratum (adjusted) odds ratio equals 1.

n
The total number of cases in all strata.

T
The number of strata (tables).

ψ
The within-stratum (adjusted) odds ratio of exposure in cases relative to controls. This odds ratio also equals the odds ratio for disease in exposed subjects relative to unexposed subjects. It approximates the within-stratum (adjusted) relative risk in case-control studies when the disease is rare in the general population.

Input Tables

Enter the proportion of cases in each table. The sum of these proportions must equal one. For each table enter the number of controls per case and the proportion of exposed controls that you anticipate in you study. For example, suppose there will be T = 2 strata with subjects divided as follows:

Unexposed ? ?
